Selamat datang di Pressure PRO

Pressure Pro adalah aplikasi web untuk analisa nilai pressure pada data menggunakan algoritma XGBoost. Aplikasi ini memungkinkan pengguna untuk melatih model gradient boosting dan mendeteksi nilai pada data.

Panduan Penggunaan

Untuk menggunakan Pressure Pro, pengguna dapat mengikuti langkah-langkah berikut:

  1. Melatih Model XGBoost dengan Randomized Search:

    Pengguna dapat mengakses menu Latih Data untuk melatih model XGBoost. Langkah-langkah untuk melatih model antara lain:

    1. Masuk ke menu Latih Data.
    2. Mulai proses pelatihan model.
    3. Tunggu hingga pelatihan selesai.
    4. Setelah pelatihan selesai maka akan menampilakan hasil pelatihan yang memuat solusi terbaik
  2. Memprediksi nilai pada Data:

    Setelah model dilatih, pengguna dapat mengakses menu Analisis untuk memprediksi nilai pada data yang dimiliki. Langkah-langkah untuk memprediksi antara lain:

    1. Masuk ke menu Analisis.
    2. Pilih model yang sudah dilatih.
    3. Unggah file data uji berformat excel (csv)
    4. Mulai proses prediksi
    5. Tunggu hingga proses selesai.
    6. Setelah proses selesai maka akan menampilakan hasil prediksi dan grafik nilai hasil prediksi
